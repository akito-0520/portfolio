## 概要

2024 年 10 月 5 日〜 6 日に開催されたハッカソンで制作した TODO アプリ。後回しにして忘れがちなタスクを管理し、通学・通勤時間などの隙間時間を有効活用することを目的としている。SNS を眺めて過ごしてしまう時間を、小さなタスク消化に置き換えるための体験を提供する。

## 技術スタック

| レイヤー       | 技術                                |
| -------------- | ----------------------------------- |
| フロントエンド | Next.js 14 / TypeScript / shadcn/ui |
| バックエンド   | Python 3.12                         |
| 通知           | Web Push (VAPID)                    |
