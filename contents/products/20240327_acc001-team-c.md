## 概要

Asian Challenge Cup 第 1 回（2024/03/25〜2024/03/27 開催）のチーム C として制作した Web アプリ。店舗情報とレビュー、ユーザー登録を組み合わせたビンゴ形式の体験を提供し、作ったビンゴ・達成したビンゴ・保存したビンゴなどを管理できる。

担当は **フロントエンド**。React + TypeScript で UI を実装し、バックエンド（Python / AWS Lambda）が提供する REST API を通してアカウント・ビンゴ・レビュー・店舗情報などの画面を組み立てた。作品は AWS Amplify でホスティング。

## 技術スタック

| レイヤー       | 技術                        |
| -------------- | --------------------------- |
| フロントエンド | React / TypeScript          |
| バックエンド   | Python / AWS Lambda         |
| インフラ       | AWS (Amplify / Lambda ほか) |
| コンテナ       | Docker                      |
