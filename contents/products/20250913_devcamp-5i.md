## 概要

DevCamp 参加者がグループを作成・参加し、ハッカソンのチームマッチングを効率的に行うための Web アプリケーション。ルームグリッド上でメンバーをドラッグ&ドロップすることで、視覚的にチーム編成を行える点が特徴。

- Firebase Authentication によるユーザー認証とプロフィール管理
- グループの作成・編集・参加（最大 35 名）
- ハッカソンイベントの作成と、フロントエンド / バックエンド人数の指定
- ドラッグ&ドロップによるチーム編成と、リアルタイムでの変更保存

## 技術スタック

| レイヤー       | 技術                                                     |
| -------------- | -------------------------------------------------------- |
| フロントエンド | React 19 / React Router 7 / TypeScript / Tailwind CSS v4 |
| 状態管理       | Zustand                                                  |
| UI             | @dnd-kit（ドラッグ&ドロップ）                            |
| バックエンド   | Firebase Functions / Firestore / Firebase Authentication |
| ビルド         | Vite                                                     |
| インフラ       | Vercel / Firebase                                        |
