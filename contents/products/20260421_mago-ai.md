## 概要

シニア（おばあちゃん）向けに、iPhone / スマートフォン操作の質問に LINE で答える Bot アプリ。お孫さんが管理画面でユーザーを登録し、おばあちゃんが LINE で質問すると Claude が優しく回答する設計。

- **LINE Bot**：質問すると Claude が iPhone 操作の手順をやさしく回答（1 回 150〜250 文字）
- **管理画面**：Google ログイン → 6 桁の登録コードを発行 → 紐付けユーザー管理
- **会話ログ閲覧**：管理画面からユーザーごとの会話履歴を確認
- **セッション管理**：Rich Menu の「新しい質問をはじめる」で会話履歴をリセット

## 技術スタック

| -------------- | ---------------------------------------------------------- |
| バックエンド   | Go 1.26 + Echo + sqlx                                      |

| インフラ       | Fly.io（バックエンド）/ Vercel（フロントエンド）/ Supabase |
| CI/CD          | GitHub Actions（lint + test + gitleaks + 自動デプロイ）    |
