## 概要

Next.js App Router で構築したランディングページ。アニメーションを活用し、ブランドの世界観を表現することを目的とした制作物。Vercel Analytics を組み込み、公開後のアクセス状況を継続的に計測している。

## 技術スタック

| レイヤー       | 技術                                                 |
| -------------- | ---------------------------------------------------- |
| フロントエンド | Next.js 16 / React 19 / TypeScript / Tailwind CSS v4 |
| アニメーション | Framer Motion                                        |
| 解析           | Vercel Analytics                                     |
| インフラ       | Vercel                                               |
