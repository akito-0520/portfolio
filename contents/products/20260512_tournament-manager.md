## 概要

石川工業高等専門学校の球技大会で使用するトーナメント表リアルタイム表示 Web アプリ。球技大会運営から提供される外部 API を SWR で 60 秒ポーリング取得し、ブラケット形式で表示する。

- 8 競技（バレーボール・サッカー・バスケットボール・ボッチャ・レスリング・ゴルフ・ソフトボール・ダーツ）のタブ切替
- ブラケット形式のトーナメント表示（横スクロール対応）
- 自動 / 手動ポーリング切替 + 最終更新時刻表示
- 試合詳細モーダル

## 技術スタック

| レイヤー       | 技術                                               |
| -------------- | -------------------------------------------------- |
| フロントエンド | Next.js 16 + TypeScript + shadcn/ui + Tailwind CSS |
| データ取得     | SWR（60 秒ポーリング）                             |
| インフラ       | Vercel                                             |
