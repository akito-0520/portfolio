## 概要

PHP と MySQL を用いたサーバーサイド Web 開発の基礎を学ぶために制作した、マッチング・チャット機能を備えた Web サイト。会員登録・ログイン認証、趣味の選択によるマッチング、ユーザー間のチャット、統計情報表示など、ひと通りの Web アプリケーションの流れを一人で組み上げた個人開発の初期作品。

## 技術スタック

| レイヤー       | 技術                    |
| -------------- | ----------------------- |
| バックエンド   | PHP                     |
| フロントエンド | HTML / CSS / JavaScript |
| データベース   | MySQL                   |
