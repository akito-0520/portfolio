## 概要

React と TypeScript に入門するために制作した、四則演算対応のシンプルな電卓アプリ。コンポーネント分割と状態管理の基礎を学ぶことを目的とした制作物。

## 技術スタック

| レイヤー       | 技術                            |
| -------------- | ------------------------------- |
| フロントエンド | React 18 / TypeScript / CSS     |
| ビルド         | Vite / @vitejs/plugin-react-swc |
| 品質管理       | ESLint                          |
